# BuildFlow - Construction Management System

BuildFlow is a comprehensive web-based platform designed to manage construction projects, advance payments, and material tracking efficiently.

## Tech Stack
* **Backend:** Java Spring Boot
* **Frontend:** React.js / Vite
* **Database:** MySQL
* **Other:** Spring Security, Maven

## Key Features
* Advance payment management logic.
* Project progress tracking.
* User roles and authentication.
* 3D model integration for housing designs.

## How to Run
1. Clone the repository.
2. Configure MySQL database in `application.properties`.
3. Run the Spring Boot application.
